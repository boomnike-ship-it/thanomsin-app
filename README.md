# THANOMSIN ENGINEERING - Web App
ระบบบริหารงานบริษัท THANOMSIN ENGINEERING (ฟรี 100%)

## ✨ Features
- 🔐 Authentication (Email/Password)
- 📊 Dashboard (KPI Cards + Charts)
- 💰 Expense Management
- 📄 Invoice Management
- 🏗️ Project Tracking
- 👥 Customer Management
- 🧾 Withholding Tax (WHT)
- 🌐 Multi-language (TH/EN)

## 🔧 Tech Stack
- **Frontend:** HTML + Vanilla JS + Tailwind CSS
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Netlify (Free)
- **Authentication:** Supabase Auth

## 🚀 Quick Start

### 1. Setup Supabase
- สร้าง Supabase Account: https://supabase.com
- สร้าง Project ชื่อ `thanomsin-db`
- Copy `Project URL` + `Anon Key`

### 2. Deploy to Netlify
- เปิด https://netlify.com
- Sign in with GitHub
- New Site from Git → เลือก `thanomsin-app`
- Deploy

### 3. Add Supabase Config
- เปิด Web App
- ไปที่ Settings
- Paste Supabase URL + Anon Key
- Save
